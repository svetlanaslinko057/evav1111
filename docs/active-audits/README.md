# 🔴 Active Audits — DO NOT CLOSE UNTIL 100% RESOLVED

> **Эта папка содержит активные аудиты с открытыми пунктами.**
> Документы здесь **обязательны к проверке в начале каждой сессии**, пока статус не станет `✅ 100% CLOSED`.

## Правила

1. **Не удалять файлы из этой папки**, пока в шапке документа стоит `Status: 🔴 ACTIVE` или `🟡 IN PROGRESS`.
2. **При закрытии каждого пункта** обновлять чек-лист в самом документе (раздел «Progress Tracker»).
3. **Финальный закрывающий PR** должен:
   - перевести `Status:` → `✅ CLOSED` и зафиксировать дату;
   - переместить файл из `active-audits/` в `closed-audits/<YYYY-MM>/`.
4. **Любой агент / разработчик**, открывающий сессию по этому проекту, **обязан**:
   - прочитать `/app/memory/active_issues.md`;
   - проверить незакрытые пункты в файлах этой папки;
   - не запускать новые фичи поверх блокирующих обрывов (P1).
