## 7. Findings (non-blocking)

### F1 — `sentence-transformers` отсутствует
**Лог:**
```
ERROR - Embedding error for template Online Marketplace: No module named 'sentence_transformers'
ERROR - Embedding error for template SaaS Dashboard: ...
ERROR - Embedding error for template Fitness & Wellness App: ...
ERROR - Embedding error for template E-Commerce Store: ...
```
**Impact:** Низкий. 4 scope-template embeddings падают на старте, fallback на keyword-matching работает.
**Fix:** Добавить `sentence-transformers` в `backend/requirements.txt` (~150 MB зависимостей; по необходимости).

### F2 — `john@atlas.dev` / `john123` → 401
**Симптом:** Один из 4 quick-access логинов не работает (admin/client/tester — OK).
**Причина:** Drift в `mock_seed.py` — пароль для `john` не совпадает с документированным.
**Impact:** Низкий. Альтернативные тестовые аккаунты есть.
**Fix:** `backend/mock_seed.py` → seed users → выставить `john@atlas.dev` пароль `john123` (либо поменять в документации).

### F3 — Duplicate Operation IDs (OpenAPI warnings)
**Лог:** `UserWarning: Duplicate Operation ID change_role_api_admin_users_v2__user_id__role_post` и ~10 аналогов в `admin_users_layer.py`, `server.py`, `legal_contract_layer.py`.
**Impact:** Низкий. Только OpenAPI schema warning — endpoints работают, но client codegen может путаться.
**Fix:** Уникализировать имена функций или указать `operation_id=` явно в декораторах.

### F4 — Rate-limit агрессивный на `/api/auth/login`
**Лог:** `ratelimit 10 per 1 minute (127.0.0.1) exceeded` (получили при smoke).
**Impact:** При активном тестировании быстро упирается в 10 запросов / минуту с одного IP.
**Fix:** Поднять лимит до 30/мин для dev/preview окружения через env.

### F5 — Web styling warnings (только web preview)
```
Invalid style property of "borderColor". Value is "var(--t-primary)44"...
"shadow*" style props are deprecated. Use "boxShadow".
props.pointerEvents is deprecated. Use style.pointerEvents
```
**Impact:** Только в web-сборке Expo. На native не влияет.
**Fix:** Заменить `shadow*` → `boxShadow`, `pointerEvents` prop → `style.pointerEvents`, hex+alpha строки.

### F6 — `expo-notifications` web warning
```
[expo-notifications] Listening to push token changes is not yet fully supported on web.
```
**Impact:** Только web. На native — работает штатно.

**Impact:** Средний. `GET /api/web-ui/` сейчас отдаёт 200 (вероятно, fallback или встроенный шаблон), но полный admin web UI может не работать.
**Fix:** Если admin SPA нужен — восстановить из прошлой сессии (`audit/AUDIT_2026-05-29_REDEPLOY_7676767_E1_RU.md` это упоминала) или из бэкапа.
