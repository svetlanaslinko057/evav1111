## 6. Findings (non-blocking)

### F1 — `sentence-transformers` отсутствует
**Лог:**
```
ERROR - Embedding error for template Online Marketplace: No module named 'sentence_transformers'
ERROR - Embedding error for template SaaS Dashboard: ...
ERROR - Embedding error for template Fitness & Wellness App: ...
ERROR - Embedding error for template E-Commerce Store: ...
```
**Impact:** Низкий. 4 scope-template embeddings падают на старте, но fallback на keyword-matching работает.
**Fix:** Добавить `sentence-transformers` в `backend/requirements.txt` (~150 MB зависимостей; решение по необходимости).

### F2 — `john@atlas.dev` / `john123` → 401
**Симптом:** Один из 5 quick-access логинов не работает (admin/client/tester/multi — OK).
**Причина:** Drift в `mock_seed.py` — пароль для `john` не совпадает с документированным (документировался как `john123` в прошлых сессиях).
**Impact:** Низкий. Альтернативные тестовые аккаунты есть (admin/client/tester покрывают все основные роли).
**Fix:** Открыть `backend/mock_seed.py`, найти секцию seed users, выставить `john@atlas.dev` пароль на `john123` (или поменять в документации). Сразу не правлю — пользователь просил аудит, не правки.

### F3 — Duplicate Operation IDs (OpenAPI warnings)
**Лог:**
```
UserWarning: Duplicate Operation ID change_role_api_admin_users_v2__user_id__role_post
UserWarning: Duplicate Operation ID logout_all_api_admin_users_v2__user_id__logout_all_post
UserWarning: Duplicate Operation ID soft_delete_api_admin_users_v2__user_id__delete
```
**Impact:** Низкий. Только OpenAPI schema warning — endpoints работают, но client codegen может путаться.
**Fix:** Уникализировать имена функций в `backend/admin_users_layer.py` или указать `operation_id=` явно.

### F4 — Rate-limit агрессивный на `/api/auth/login`
**Лог:** `ratelimit 10 per 1 minute (127.0.0.1) exceeded`
**Impact:** При активном тестировании быстро упирается в 10 запросов / минуту с одного IP.
**Fix:** Можно поднять лимит до 30/мин для dev/preview окружения через env.

### F5 — Web styling warnings (только web preview)
```
Invalid style property of "borderColor". Value is "var(--t-primary)44"...
"shadow*" style props are deprecated. Use "boxShadow".
props.pointerEvents is deprecated. Use style.pointerEvents
```
**Impact:** Только в web-сборке Expo. На native не влияет.
**Fix:** Заменить `shadow*` → `boxShadow`, `pointerEvents` prop → `style.pointerEvents`, исправить hex+alpha строки (`color44` → `color` с opacity).

### F6 — `expo-notifications` web warning
```
[expo-notifications] Listening to push token changes is not yet fully supported on web.
```
**Impact:** Только web. На native — работает штатно. Push notifications требует boilerplate с Firebase + native build.
